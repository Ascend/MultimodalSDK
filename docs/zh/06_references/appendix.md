# 附录

## 软件中包含的公网地址

Multimodal SDK 安装包中的网址在安装结束后会被清除，本手册中列出的公开网址和邮箱地址，SDK 本身均不会访问，不会造成风险。

更多公网地址请参见 [Multimodal SDK公网地址.xlsx](../resource/Multimodal%20SDK公网地址.xlsx)（随 Multimodal SDK 发布文档包提供）。

## 软件通信矩阵

目前 Multimodal SDK 开发套件包不会主动打开或者依赖任意端口，因此不涉及通信矩阵。

## 错误码

| 错误码（十六进制） | 错误信息 | 处理建议 |
| -- | -- | -- |
| 0x10100001 | 参数无效 | 检查输入参数类型、取值范围及必填项是否与 API 文档一致。 |
| 0x10100002 | 不支持的类型 | 确认输入数据类型在 API 支持列表内（如 dtype、图像格式等）。 |
| 0x10100003 | 指针无效 | 检查传入对象是否已正确初始化，避免使用已释放或空对象。 |
| 0x10100004 | 值超出范围 | 核对参数约束（如图像尺寸 [10, 8192]、权限 640 等）是否满足要求。 |
| 0x102003E9 | 内存分配失败 | 检查系统可用内存，减少并发或批量处理规模后重试。 |
| 0x102003EA | 内存拷贝失败 | 确认源数据内存连续且有效，检查数据大小是否超出限制。 |
| 0x102003EB | 内存释放失败 | 检查是否存在重复释放或内存损坏，必要时重启进程。 |
| 0x102003EC | 系统内存不足 | 释放不必要的内存占用，或增加系统可用内存后重试。 |
| 0x102003ED | 打开文件失败 | 确认文件路径有效、文件存在且未被其他进程锁定。 |
| 0x102003EE | 文件权限无效 | 确保文件 owner 为当前用户，且权限不高于 640。 |
| 0x102003EF | 文件大小无效 | 检查文件大小是否在 API 限制范围内（如图像不超过 1GB）。 |
| 0x103007D1 | FFmpeg 初始化失败 | 确认 FFmpeg 依赖已正确安装，检查视频文件格式是否为 mp4。 |
| 0x103007D2 | Libjpeg 初始化失败 | 确认 Libjpeg 依赖可用，检查系统库路径配置。 |
| 0x103007D3 | Libjpeg 读取文件失败 | 确认图像为 jpg/jpeg 格式且文件未损坏，检查文件权限。 |
| 0x103007D4 | FFmpeg 执行失败 | 检查视频文件完整性、分辨率是否在 [480, 4096] 范围内。 |
| 0x10400BB9 | 内部算子失败 | 检查输入数据格式与尺寸是否符合算子要求，查看详细日志定位。 |
| 0x10400BBA | 内部函数执行失败 | 查看 SDK 日志输出，确认前置步骤是否成功完成。 |
| 0x10400BBB | 内部类型转换失败 | 确认源数据与目标类型兼容（如 dtype、排布格式 NCHW/NHWC）。 |
| 0x10500FA1 | 等待超时 | 检查系统负载，适当增大超时时间或减少并发任务。 |
| 0x10500FA2 | 线程池状态无效 | 重启相关进程，避免在不正确的状态下重复提交任务。 |

## 环境变量说明

以下环境变量会在程序安装或运行时使用，请确保有效。

**表 1** 环境变量

| 环境变量名 | 说明 |
| -- | -- |
| PATH | 可执行程序的文件路径。 |
| LD_LIBRARY_PATH | 动态链接库路径。 |
| PYTHONPATH | Python 模块文件的默认搜索路径。 |
| HOME | 当前用户的家目录。 |
| PWD | 当前工作目录。 |
| TMPDIR | 临时文件路径。 |
| ASCEND_HOME_PATH | CANN 相关资源路径，请确保路径有效性。 |
| ASCEND_HOME | CANN 安装目录，请确保路径有效性。 |
| ASCEND_VERSION | CANN 版本号，请勿随意改动。 |
| ASCEND_CUSTOM_OPP_PATH | AscendC 算子部署路径，请勿随意改动。 |
| MULTIMODAL_SDK_HOME | Multimodal SDK 安装目录（`.run` 完整安装时由 `set_env.sh` 设置）。Wheel 包安装时无需配置。 |
| HF_DATASETS_OFFLINE | 离线加载 Hugging Face 数据集。 |
| HF_HUB_OFFLINE | 在离线模式下运行 Hugging Face 库。 |

> [!CAUTION] 注意
> 使用本软件提供的 patcher 运行 vllm 推理或其他开源库时，还会使用到其他开源库相关环境变量，本文不做声明，请用户自行确保环境变量真实有效。
